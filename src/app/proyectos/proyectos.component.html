SISGEMA - Sistema de Gestión de Matrículas IESTP LAREDO

SISGEMA - Sistema de Gestión de Matrículas IESTP Laredo es una plataforma web diseñada para administrar el proceso de matrícula en el Instituto de Educación Superior Tecnológico Público (IESTP) Laredo. Permite gestionar la inscripción de estudiantes, asignación de cursos, control de pagos y generación de reportes académicos y administrativos, optimizando la gestión educativa con mayor eficiencia y precisión.

Proyecto Ori Retail

Ori Retail es un sistema de gestión de medicamentos diseñado para farmacias, que permite administrar compras, ventas, proveedores y stock de productos farmacéuticos. Facilita el control de inventarios, la facturación, la trazabilidad de medicamentos y la gestión de clientes, optimizando los procesos operativos y mejorando la eficiencia en la atención al público.

MediRetail X

MediRetailX es un sistema WEB de gestión de stock de medicamentos para farmacias y establecimientos de salud. Permite administrar el inventario con registro por lote, número de registro sanitario, precios, laboratorio, unidades disponibles y fechas de vencimiento. Incluye funciones de venta con emisión de facturas y boletas, un catálogo de productos y un sistema de alertas para notificar sobre productos próximos a vencer, asegurando un control eficiente y seguro del stock.
